# Healthcare-Dashboard
📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze the impact of smoking on health across different age groups. It provides insights into patient demographics, smoking habits, and associated health risks such as cholesterol and hypertension.

The goal of this project is to help users understand patterns and correlations between smoking behavior and health conditions using data visualization.

🎯 Key Objectives


Analyze smoking status distribution (Never, Current, Former)
Evaluate health risks across age groups
Identify trends in smoking duration and daily intake
Understand relationship between cholesterol, hypertension, and smoking
Compare smoking patterns by gender

📈 Dashboard Features

📌 KPI Cards (Total Patients, Avg Age, Avg BMI)
📊 Donut Chart for Smoking Status Distribution
📉 Trend Analysis (Smoking Duration vs Daily Intake)
📊 Bar Charts for Risk Analysis (Cholesterol & Hypertension)
👥 Gender-based Smoking Insights
🎯 Interactive Filters (Health Condition Toggle)


🛠️ Tools & Technologies Used

Power BI – Data visualization & dashboard creation
Excel / CSV – Data source
DAX (Data Analysis Expressions) – Calculations & measures
Data Modeling – Relationships & transformations
🧠 Key Insights
Majority of patients fall under non-smokers, followed by current and former smokers
Higher age groups show increased cholesterol and hypertension risk
Smoking duration tends to decrease with age, while health risks increase
Noticeable variation in smoking habits between genders
