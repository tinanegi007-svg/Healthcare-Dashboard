# Healthcare-Dashboard
An interactive Power BI dashboard analyzing the impact of smoking on health across different age groups.
The report highlights key metrics such as patient count, smoking status distribution, cholesterol and hypertension risk, and behavioral patterns like smoking duration and daily intake.

The dashboard enables users to:

Compare health risks across age groups
Analyze smoking habits by gender
Identify correlations between smoking, cholesterol, and hypertension
Gain actionable insights through intuitive visualizations and KPIs
